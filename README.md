# 🎵 JS Music Player

A music player built with HTML, CSS, and JavaScript. Features a playlist view and a now-playing view with background video per song.

---

## 🚀 Features

- 🎬 **Hover Preview** — hover over a song to preview its background video
- 📱 **Two Screen Layout** — switch between the playlist and now-playing view
- 🎚 **Progress Bar** — click or drag to seek through the song
- ⏯ **Playback Controls** — play, pause, next, and previous song
- 🎚 **Progress Bar** — click or drag to seek through the song
- 🖼 **Cover Art** — each song displays its own album cover
- 🎥 **Background Video** — each song has a matching background video

---

## 🧠 What I Learned

- How `display: none` and `display: block` work to switch between screens without real navigation
- How `onmouseenter` and `onmouseleave` trigger hover events in HTML
- Why `bgVideo.pause()` and `bgVideo.src = ""` are both needed to fully clear a video
- How HTML, CSS, and JS all connect through **class names**
- How JS code only runs when something **triggers it** (like a click or hover) — not automatically on page load
- How to use a `isPlaying` flag to guard against unwanted behavior

## ▶️ How to Run

1. Download or clone the repo
2. Add your own `song1.mp3`, `cover1.jpg`, `background1.mp4` files (and so on for each song)
3. Open `index.html` in your browser
4. That's it!
